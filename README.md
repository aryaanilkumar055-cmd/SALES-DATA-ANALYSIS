# 📊 Sales Data Analysis using Python

## Project Description

This project analyzes sales data using Python. It performs exploratory data analysis (EDA) to calculate total sales, identify the best-selling product, analyze monthly and category-wise sales, and visualize business insights using charts.

## Features

* Reads sales data from an Excel file
* Displays the sales dataset
* Calculates total sales revenue
* Creates a new **Sales** column using Quantity Sold × Price
* Identifies the best-selling product
* Analyzes monthly sales performance
* Analyzes category-wise revenue
* Displays:
  * Line chart of monthly sales trend
  * Bar chart of product-wise sales
  * Pie chart of category-wise revenue contribution

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Microsoft Excel

## Files

* `sales.py` - Main Python program
* `SalesDataAnalysis.xlsx` - Sales dataset

## Dataset Columns

* Product
* Category
* Month
* Quantity_Sold
* Price

## Output

The program generates:

* Sales dataset with calculated Sales column
* Total sales revenue
* Best-selling product
* Monthly sales summary
* Category-wise sales summary
* Monthly Sales Trend (Line Chart)
* Product-wise Sales (Bar Chart)
* Category-wise Revenue Contribution (Pie Chart)

## Note

This project requires the `SalesDataAnalysis.xlsx` file to run. Make sure the dataset path in the Python script matches the location of the Excel file on your system.
